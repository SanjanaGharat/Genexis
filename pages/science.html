Science Behind It

Epigenetic Aging Methodology

DNA Methylation Clocks

Sample Collection

Whole blood samples collected and processed using standardized protocols to isolate DNA while preserving methylation patterns.

Array Processing

DNA processed using Illumina EPIC arrays measuring methylation at ~850,000 CpG sites across the genome.

Data Normalization

Raw data processed using Noob normalization and BMIQ correction to account for technical variability.

Clock Application

Multiple epigenetic clocks (Horvath, Hannum, PhenoAge, GrimAge) applied to estimate biological age.

Technical Specifications

Platform

Illumina EPIC Array

CpG Sites

850,000+

Accuracy

±2.7 years

Precision

±0.5 years

Statistical Methods

Algorithm Details

Model Type

Elastic Net Regression

CpG Sites

353 (Horvath)

Training Set

n=8,000

Validation

10-fold CV

The epigenetic clocks were trained using elastic net regression (α=0.5) on large datasets of DNA methylation data paired with chronological age or clinical biomarkers.

Feature selection was performed to identify CpG sites most predictive of aging outcomes, with weights optimized to minimize prediction error in cross-validation.

Comparison of clock performance in predicting mortality risk (AUC-ROC)

Limitations & Future Directions

Current Limitations

  • Tissue specificity of methylation patterns
  • Limited longitudinal data for intervention effects
  • Variability in sample collection methods
  • Ethnic diversity in training datasets
  • Cost and accessibility of methylation arrays

Future Enhancements

  • Single-cell methylation profiling
  • Integration with other omics data
  • Improved clocks for specific tissues
  • Real-time methylation monitoring
  • Open-source analysis pipelines
